# 🛒 Taaja Bazaar – Grocery Store E-Commerce Website

A modern and responsive Grocery Store E-Commerce Website built using **HTML, CSS, and JavaScript**. The project provides an interactive online shopping experience where users can browse grocery items, add products to their cart, manage orders, and simulate secure online payments.

## 🌟 Features

- 🏠 Modern Homepage
- 🔍 Product Search
- 🛍️ Product Categories
- ❤️ Wishlist
- 🛒 Shopping Cart
- ➕ Add & Remove Products
- 📦 Order Management
- 💳 Simulated eSewa & Khalti Payment
- 📱 Fully Responsive Design
- 👤 User Authentication (Frontend)
- ⭐ Product Ratings & Reviews
- 🎟️ Coupon System
- 📍 Order Tracking
- 🛠️ Admin Dashboard
- 📊 Sales Statistics
- 📦 Inventory Management
- 🎨 Beautiful UI/UX

---

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage (Database Simulation)
